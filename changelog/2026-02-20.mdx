## Documentation Site Launch

[docs.limitguard.ai](https://docs.limitguard.ai) is now live.

<Note>
  The LimitGuard API has been live and in use prior to this documentation launch. These docs formalize what was previously available only to early partners.
</Note>

### What's included

**API Reference**
- Complete endpoint reference covering all 64 endpoints
- Interactive API playground for testing requests directly in the browser
- Multi-language code examples in curl, Python, and JavaScript

**Guides**
- Error codes and handling patterns
- Rate limits and retry strategies
- Webhook setup and event reference
- Trust architecture overview
- AI agent integration patterns
- Sandbox mode and test credentials
- x402 protocol implementation guide
